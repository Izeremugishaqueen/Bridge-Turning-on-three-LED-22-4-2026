🚦 Traffic Light System with 3 LEDs

This project simulates a basic traffic light system using three LEDs representing the standard traffic signal colors: Red, Yellow, and Green. It is a beginner-friendly electronics and programming project, commonly implemented using a microcontroller like Arduino.

🔧 Components Used
Microcontroller (e.g., Arduino Uno)
Controls the sequence and timing of the LEDs.
Red LED 🔴
Represents the "Stop" signal.
Yellow LED 🟡
Represents the "Caution" or "Wait" signal.
Green LED 🟢
Represents the "Go" signal.
Resistors (220Ω – 330Ω)
Protect the LEDs from excessive current.
Breadboard
Used to assemble the circuit without soldering.
Jumper Wires
Connect components together.
USB Cable / Power Source
Supplies power to the microcontroller.
⚙️ How It Works

The microcontroller is programmed to turn on the LEDs in a sequence that mimics real traffic lights:

Green LED turns ON (vehicles can go)
Yellow LED turns ON (prepare to stop)
Red LED turns ON (vehicles must stop)

This cycle repeats continuously with delays between each state.

🎯 Purpose of the Project
Learn basic electronics and circuit connections
Understand how LEDs work
Practice programming with microcontrollers
Simulate real-world systems in a simple way
